# Mi Portafolio - Landing Page 🚀

Un proyecto de landing page personal moderno y responsivo, diseñado para mostrar mis habilidades como desarrollador web, mis proyectos recientes y mi experiencia en diseño frontend.

## 🌟 Características Principales

- **Diseño Responsivo:** Completamente adaptable a teléfonos móviles, tablets y monitores de escritorio empleando Flexbox, CSS Grid y Media Queries.
- **Tipografía Moderna:** Integración de la fuente [Inter](https://fonts.google.com/specimen/Inter) de Google Fonts para una lectura cómoda y estética premium.
- **Micro-interacciones y UI:** Botones dinámicos que se invierten al pasar el cursor y paleta de colores cuidadosamente seleccionada para resaltar la información.
- **Animaciones Suaves (Fase 1):** Utiliza la API nativa de `IntersectionObserver` de JavaScript para animar la aparición de secciones (`fade-in` y `slide-up`) y galerías en cascada a medida que se hace scroll.
- **Efecto de Máquina de Escribir (Fase 2):** Integración de [Typed.js](https://github.com/mattboldt/typed.js/) para rotar frases descriptivas dinámicas en la cabecera (Hero Section).
- **Parallax y Contadores (Fase 3):** Efecto de fondo con ilusión de profundidad (parallax) al bajar por la página, e interacciones de números que ascienden de cero a su meta cuando la sección es visible en la pantalla.

## 🛠️ Tecnologías Usadas

- **HTML5:** Estructura semántica.
- **CSS3:** Variables nativas (`:root`), Flexbox, CSS Grid, animaciones (`transform`, `transition`), y técnicas modernas de diseño.
- **JavaScript (Vanilla):** Lógica del DOM, IntersectionObserver y cálculos matemáticos para efectos visuales avanzados.
- **Bibliotecas externas:** [Typed.js](https://github.com/mattboldt/typed.js/) (vía CDN) para la animación de texto.

## 🚀 Instalación y uso

Este proyecto no requiere instalación de dependencias pesadas ni compiladores. Para verlo y modificarlo:

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Firefox, Edge, Safari).
3. (Opcional) Si deseas editar el código, puedes abrir esta carpeta en tu editor de código favorito (ej. VS Code) y usar herramientas como la extensión "Live Server" para ver los cambios en tiempo real.

## 📂 Organización del Proyecto

- `index.html`: La estructura principal del sitio.
- `style.css`: Toda la magia visual y diseño adaptativo.
- `animations.js`: El motor interactivo detrás de los contadores, parallax y aparición suave.
- `/images/`: Carpeta que almacena las fotografías e ilustraciones usadas en el diseño.
